# InfotechChat

Тестове завдання для компанії `"ІНФОТЕХ"`, `ЧАТ` на мові програмування `Elixir`, що написаний за допомогою бібліотек `n2o` та `nitro`.

# Підняття проекту локально

- Збираємо через `Dockerfile` та `docker-compose`.
    - Виконайте команду `docker-compose up --build` або також може бути `docker compose up --build`, в залежності від вашого Docker середовища.
    - Буде зібрано та запущено локально `production ready` реліз проекту.

- Звичайний сценарій запуску проекту локально.
    - Середовище - Якщо ви використовуєте `asdf` для контролю версій мов програмування `Elixir` та `Erlang` виконайте команду `asdf install`, вона встановить всі потрібні версії мов програмування, що використвуються в даному проекті. Якщо ж ви використовуєте інший спосіб то встановіть версію `Elixir` `1.16.1` а також вевсію `Erlang` `26.2.2` в ручному режимі.
    - Залежності - `mix deps.get`.
    - Компіляція - `mix deps.compile && mix compile`
    - Запуск проекту - `iex -S mix`.

# Використання

- Відкрийте `http://localhost:8004/index.html`
